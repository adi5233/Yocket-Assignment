## Cloning the repository

```shell
git clone https://github.com/adi5233/Yocket-Assignment.git
```

## Install packages

```shell
npm install
```

## To start the dev server

```shell
npm run dev
```

## Go to client folder

```shell
cd client
```

## Install packages

```shell
npm install
```

## To start the client server

```shell
npm start
```
